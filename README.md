# ![Image](https://github.com/user-attachments/assets/ef04cb1f-406a-4bfe-91c6-1d892d7e9a93) TRIPLY (트립리)
> 한국수출입은행의 환율 API를 이용해 최신 환율로 간편하게 만드는 나만의 한 달 살기 예산 플래너

<br>

## 📌 서비스 개요
TRIPLY는 해외 한 달 살기를 준비하는 사용자를 위한 **예산 플래너 서비스**입니다.

여행을 준비하는 과정에서 번거로웠던 환율 계산과 예산 설계를 통합하여

사용자가 **카테고리별 예산을 입력하면 실시간 환율이 적용된 합계**를 바로 확인할 수 있습니다.

<br>


## 🎯 기획 의도
- 해외 한 달 살기를 준비할 때, 환율 변동과 예산 계산 과정에서 겪는 불편함을 해소하기 위해 기획
- **실시간 환율 API**를 활용하여 최신 환율이 자동으로 반영되는 정확하고 직관적인 예산 계산 기능 구현

<br>

## ✨ 주요 기능
### 1. 실시간 환율 계산
- **한국수출입은행의 현재환율 API**를 연동하여 최신 환율 데이터를 자동으로 반영
- 사용자가 입력한 금액을 **원화 ↔️ 현지 통화** 기준으로 즉시 변환

### 2. 예산 입력 및 합계 계산
- 숙소, 식비, 교통, 관광, 쇼핑 등 카테고리별 금액 입력
- 변경 시마다 실시간 합계 자동 계산 및 반영

### 3. 예산 결과 요약
- 카테고리별 비율 및 총합을 테이블 형태로 정리하여 한눈에 파악 가능

### 4. ‘내 플랜’ 저장 기능
- 사용자가 입력한 예산 데이터를 **로컬 스토리지에 저장 및 불러오기** 가능
